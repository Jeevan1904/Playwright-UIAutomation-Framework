# Playwright-UIAutomation-Framework
UI Automation framework using playwright - Learning and POC

Features to be included:

* Record and play [Codegen]
* Page Object Models
* Project Structure
* Pytest Framework
    * markers
* Reports
* Parallel execution
* Pytest Fixtures
    * setup fixture
    * teardown fixture
    * scoping
* Screenshots
* Video Recordings
* Tracing and Debugging
* Datadriven testing
* CI/CD Integration
